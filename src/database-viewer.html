Database Viewer

The Database Viewer is a web application designed to provide a user-friendly interface for interacting with various types of databases. This application allows users to view and manage databases, tables, and records with ease. Key features include database selection, table viewing, and the ability to edit records and column types. The system supports multiple database types, including MySQL, SQL Server, and PostgreSQL, ensuring versatility and broad compatibility.

  • PHP: The server-side scripting language used to build the application.
  • PDO: The PHP Data Objects extension used for database interactions.
  • MySQL, SQL Server, PostgreSQL: The supported database engines.
  • Bootstrap 5: The front-end framework used for responsive and modern user interface design.
  • XAMPP: The development environment used to run the application locally.
  • Adminer: An alternative database management tool that can be used alongside the application.
