Healthcare Management System

The Healthcare Management System is a web application designed to streamline the administrative and clinical processes within healthcare facilities. This application allows healthcare providers to efficiently manage patient records, appointments, medical records, billing, and doctor information. Key features include patient and doctor management, appointment scheduling, and detailed medical record-keeping. The system supports different user roles, including administrators and doctors, with appropriate access rights to ensure data security and integrity.

  • ASP.NET Core: The web application framework used to build and run the application.
  • Entity Framework Core: The ORM (Object-Relational Mapper) used for database operations.
  • SQLite: The database engine used for data storage.
  • Bootstrap: The front-end framework used for responsive and modern user interface design.
  • Razor Pages: The view engine used for rendering dynamic web pages.
  • Visual Studio Code: The IDE (Integrated Development Environment) used for application development and debugging.
