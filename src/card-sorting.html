Card Sorting App

Card Sorting App is a web application that helps teachers in creating teaching questions based on sorting cards. The web app allows students to sort cards into the right categories. Teachers can see results and statistics based on the submitted sorts by students.

  • Python
  • JavaScript
  • Flask
  • Jinja2
  • SQLAlchemy (flask-sqlalchemy)
  • Bcrypt (flask_bcrypt)
  • LoginManager (flask_login)
